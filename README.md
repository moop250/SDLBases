# SDL Basics Learning Project

Welcome to my SDL (Simple DirectMedia Layer) Basics Learning Project! This repository contains my efforts to learn the basics so that I am able to make more complex and capable graphical apps in the future.

## Objectives

- Understand the basics of setting up and using SDL.
- Learn how to create basic windows and handle events.
- Explore rendering techniques and basic graphics with SDL.
- Implement simple audio playback and other multimedia features.

## Getting Started

To get started with the project, you'll need to have SDL installed on your machine. Follow the steps below to set up your environment:

### Prerequisites

- **SDL3**: Ensure you have SDL3 installed. You can download it from the [official SDL website](https://www.libsdl.org/).
- **C/C++ Compiler**: Make sure a suitable C/C++ compiler is installed (e.g., GCC for Linux, Clang for macOS, or MinGW for Windows).
- **CMake**: This repo uses CMake to compile the source files

### Installation

1. Clone this repository to your local machine:

   ```sh
   git clone https://github.com/yourusername/SDL-Basics-Learning.git
   ```

2. Navigate to the project directory:

   ```sh
   cd SDL-Basics-Learning
   ```

3. Compile the example code:

   ```sh
   mkdir build
   cd build
   cmake ..
   make
   ```

4. Run the compiled program:

   ```sh
   ./example
   ```
